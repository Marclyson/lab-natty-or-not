# Descubra as Origens da Análise de Dados

## 📒 Descrição
O projeto conta uma curiosidade das origens da análise de dados e dá um exemplo de uma ferramenta de muito usada atualmente por profissionais dessa área

## 🤖 Tecnologias Utilizadas
> ChatGPT
> ElevenLabs
> Descript

## 🧐 Processo de Criação
O processo inicia na criação do roteiro no chatGPT. Com a estrutura formada, usei o EvenLabs para criação da voz e o Descript para inserção dos das legendas e efeitos de ondas do podcast
